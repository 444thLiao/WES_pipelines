Some useful but not routine analysis

haplotype分析

Located Whole_pipelines/aft_pipelines_analysis/haplotype/

基本上你只需要使用 python Whole_pipelines/aft_pipelines_analysis/haplotype/search_possible.py snvs.csv tumor.b_recal.bam output.csv

基本的原理，即使用csv中所指定的位点，回到bam文件中找到对应的reads，并建立起一个大的database，然后尝试在每个位点间寻找相同的reads，从而可以判断两个相近（小于reads的长度）的位点是否是在一个haplotype上，从而可以得出一定的结论。

由以上的原理也可以看出，该方法十分的原始和粗暴。所以能够识别的haplotype也十分的有限，所以需要更为有效的方法。例如gatk中calling过程中所使用的HMM方法。

Just a idea

建立样本-位点-位点的深度信息百分比，作为一个表格，然后与表型进行关联，或者说进行marker位点的分析。但是由于该表格是典型的features远远大于样本数目的数据，所以难以使用机器学习来进行特征筛选，需要引入别的方法。不然过拟合是很简单的事情。
